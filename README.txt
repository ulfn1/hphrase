How to run the calculations

1. Install anaconda3
2. Create folder 'hphrase' and go to the created folder
3  Download https://github.com/ulfn1/hphrase/tree/main
3. unzip hphrase-main.zip and move the files in 'hphrase-main' to folder 'hphrase'
4. Create conda environment 'hphrase' by command: conda env create --name hphrase --file hphrase.yml
5. In folder 'hphrase' untar the tarball: tar xvf hphrase.tar.gz
6. Download https://github.com/f48r1/TISBE/tree/main and unzip TISBE-main.zip
7. run hphrase_cmds.prc (using htest.data file)
8. The output data should now be available to collect and examine
